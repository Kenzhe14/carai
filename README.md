# 🚘 AI Car Damage Detector

Этот проект представляет собой систему машинного зрения для обнаружения автомобильных повреждений. В основе лежит модель YOLOv8 и современный веб-интерфейс на базе Streamlit с FastAPI бэкендом.

## 📦 Установка зависимостей

Убедитесь, что у вас установлен Python (3.9+). Для установки всех библиотек выполните:
```bash
python -m venv venv
source venv/bin/activate  # Для Windows: venv\\Scripts\\activate
pip install -r requirements.txt
```

## 🚀 Как запустить? (В один клик)

Для удобства добавлены скрипты авто-запуска, которые стартуют и backend, и frontend одновременно.

### Для macOS и Linux
Откройте терминал в папке проекта и выполните:
```bash
chmod +x run.sh
./run.sh
```

### Для Windows
Просто дважды кликните по файлу `run.bat` или запустите его в терминале:
```cmd
run.bat
```

## 🛠 Ручной запуск (классический)

1. Запустите бэкенд (FastAPI):
   ```bash
   uvicorn main:app --reload
   ```

2. В новом окне терминала запустите фронтенд (Streamlit):
   ```bash
   streamlit run streamlit_app.py
   ```

## 📸 Скриншоты и возможности

- **Современный UI**: Темная тема с градиентными кнопками и неоновыми стилями.
- **Моментальный анализ**: Модель анализирует фото и выдает точный вердикт с отрисовкой рамок на повреждениях.
